## 📸 Screenshots

Para obtener screenshots de alta calidad:

### Dashboard
```powershell
# Ejecutar en Windows Terminal con tema oscuro
phylax run
# Capturar con Win+Shift+S o herramienta de screenshot
```

### Bloqueo en acción
```powershell
# Tener el dashboard en un panel y un agente de IA en otro
# Ejecutar un prompt que intente acceder a un archivo protegido
# Capturar el momento exacto del BLOCK en rojo
```

### Especificaciones para screenshots

- **Resolución mínima:** 1920×1080
- **Formato:** PNG (sin pérdida)
- **Fondo:** Terminal oscuro (`#101312`)
- **Sin información personal visible** (sanitizar rutas, nombres, keys)

## 🤝 Usage Guidelines

- ✅ Usar el logo y nombre "Phylax" para contenido educativo, tutorials, reviews
- ✅ Modificar los colores para adaptarlos a tu estilo de thumbnail
- ✅ Traducir el contenido a cualquier idioma
- ❌ No usar el branding para contenido que promueva malware o actividades maliciosas
- ❌ No modificar el logo de manera que distorsione su reconocimiento
- ❌ No sugerir que Phylax es un producto de pago (es y siempre será gratis)
